Inline CSS vs. External CSS

Inline CSS

Inline style används för att styla ett specifikt HTML-element, vilket innebär att man adderar attributet style i respektive HTML-element i dokumentet.

Fördelar:

  • Effektivt när man enbart vill styla ett enskilt element
  • Användbart när man på ett snabbt sätt vill se hur en styling påverkar webbsidan
  • Man behöver inte skapa och ladda upp ett enskilt dokument

Nackdelar:

  • Att addera CSS-regler till varje element är tidskrävande och gör HTML-dokumentet rörigt
  • Att styla många element kan påverka sidans storlek och nedladdningstid

External CSS

External style innebär att man länkar sin webbsida med en extern CSS-fil. Genom att redigera i en CSS-fil kan man ändra hela webbsidan på en gång.

Fördelar:

  • Effektivt när man ska styla ett större projekt med flera sidor
  • När CSS-koden är i ett separat dokument blir det en renare struktur
  • Man kan använda samma CSS-fil för flera sidor

Nackdelar:

  • Webbsidorna kanske inte renderas korrekt förrän den externa CSS-filen har laddats
  • Att ladda upp eller länka till flera CSS-filer kan öka webbsidans nedladdningstid
